## üé® Customization

### Personal Information
Update the following files with your personal information:

- `src/components/Navbar.jsx` - Update name in logo
- `src/components/Hero.jsx` - Update name, title, and description
- `src/components/About.jsx` - Update about text and stats
- `src/components/Projects.jsx` - Replace with your actual projects
- `src/components/Contact.jsx` - Update contact information

### Styling
- Modify `tailwind.config.js` to change colors, fonts, and animations
- Update `src/index.css` for global styles and custom components
- Adjust glass effects, gradients, and animations as needed

### Content
- Replace placeholder project images with your actual project screenshots
- Update skills and technologies in `src/components/Skills.jsx`
- Modify the contact form to integrate with your preferred email service

## üåê Deployment

### Netlify (Recommended)
1. Build the project: `npm run build`
2. Drag and drop the `dist` folder to Netlify
3. Configure custom domain if needed

### Vercel
1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect the Vite project
3. Deploy with zero configuration

### GitHub Pages
1. Install gh-pages: `npm install --save-dev gh-pages`
2. Add deploy script to package.json
3. Run `npm run deploy`
